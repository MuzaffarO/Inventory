# Inventory System

A Spring Boot application to manage a simple store inventory. Users can:
- View available products
- Add new products
- Update existing products

## 📦 Technologies Used

- Java 17
- Spring Boot 3
- Hibernate / JPA
- PostgreSQL
- Maven
- Swagger UI
- JUnit 5 / Mockito

## 🚀 How to Run

1. **Install PostgreSQL** and create a database:
   ```sql
   CREATE DATABASE inventory_db;
   ```
